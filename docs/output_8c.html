Configure the analog outputs.

This configures which input to use for the analog outputs as well as the scale for the data

Parameters

input,theinput driving the output, used to get the min/max of the input to ensure selected range in in bounds
input_chnlthe input channel driving this output, 0-3 for the 4 analog inputs
maxdouble, max value of input to trigger 20mA
mindouble, min value of input to trigger 4mA
Returns
true on success, false on failure. Return false if: input_chnl not in range (0-3), input is not analog type, or select range of min-max is out of input range bounds
